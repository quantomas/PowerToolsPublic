
# Saubere Python-Installation unter Windows
https://docs.aws.amazon.com/de_de/cli/latest/userguide/awscli-install-windows.html

So installieren Sie Python 3.6 und pip (Windows)
    Laden Sie die ausführbare Installationsdatei für Python 3.6 Windows x86-64 von der Downloads-Seite von Python.org herunter.
    Führen Sie das Installationsprogramm aus.
    Wählen Sie Python 3.6 zu PATH hinzufügen.
    Wählen Sie Install Now aus.

Das Installationsprogramm installiert Python in Ihrem Benutzerordner und fügt seine ausführbaren Verzeichnisse Ihrem Benutzerpfad hinzu.

### AWS CLI

pip install awscli

aws --version

pip install --user --upgrade awscli

# Checks

where aws
> C:\Users\yourname\AppData\Local\Programs\Python\Python37\Scripts\aws
> C:\Users\yourname\AppData\Local\Programs\Python\Python37\Scripts\aws.cmd
